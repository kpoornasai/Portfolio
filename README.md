## 🚀 Getting Started

To run this portfolio locally:

```bash
# Clone the repository
git clone https://github.com/kpoornasai/Portfolio.git

# Navigate into the folder
cd Portfolio

# Open in browser
open index.html
```

Or simply open `index.html` in any browser — no build tools required.
